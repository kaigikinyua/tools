    Final Touches

    Registry paths

    1. Forms60_PATH: U:\FORMS
    2. FORMS60_TMP: C:\TEMP
    3. REPORTS60_PATH: U:\REPORTS; U:\FORMS
    4. REPORTS60_TMP: C:\TEMP
    5. SQLPATH: U:\SQLS
    6. LOCAL: COINSKENYA

    Desktop Shortcut

    1. Create a shortcut on the desktop
    2. Path: C:\orant\bin\ifrun60.exe bso017
    3. Name the shortcut Coins
    4. Rightclick the shortcut and set start in U:\forms
    5. Change icon to the path U:\forms\shoe.ico
    6. Apply the changes

      7. Make sure you have mapped to drive \\batapower\coins or \\192.168.15.28\coins
